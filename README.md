# TOTC ‚Äì Modern Online Learning Platform

TOTC is a modern, scalable frontend web application for an online learning platform,
designed with a glassy (glassmorphism) UI and smooth animations to deliver a premium user experience.
The project demonstrates real-world frontend architecture and state management practices.

---

##  Key Highlights
- Modern Glassmorphism (Glassy Mode)UI design
- Smooth and interactive animations using Framer Motion
- Scalable state management with Redux Toolkit
- Firebase integration for backend services
- Fully responsive, mobile-first layout
- Clean, maintainable, and modular codebase

---

##  Core Features
- Responsive and adaptive UI across all devices
- Global state management using Redux Toolkit
- Animated transitions and micro-interactions
- Reusable and well-structured React components
- Modern development workflow powered by Vite

---

## Tech Stack
- React.js (Vite)
- Redux Toolkit
- Tailwind CSS
- Firebase
- Framer Motion
- JavaScript (ES6+)

---

## UI & UX Design
- Glassy / Glassmorphism design system
- Modern color palette and visual hierarchy
- Smooth page transitions and component animations
- Focus on usability, clarity, and performance
